Akathist Hymn to Theotokos
 
 
(March 2010)

Using the tool that converts Greek HTML to LaTEX, I made an A4 booklet with the Acathist Hymn to Theotokos.

The Akathist is a hymn dedicated to a saint, holy event, or one of the persons of the Holy Trinity. This particular Akathist is dedicated to the Mother of God (Theotokos).

If you print it on an A4 printer, with "print on both sides" and "flip on short edge" enabled, you will get an easy to fold booklet with the 24 oikoi.

For those of you using an e-book reader, here is a version made for 6-inch displays (like the Sony Reader).

Finally, here is the source code - in case you want to modify it (I think the original HTML page - which I found somewhere on the web - has some errors). Contributions most welcome!
