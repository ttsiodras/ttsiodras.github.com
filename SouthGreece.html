A real-time flight over South Greece

 
(September 2003)

A demo of the OpenGL implementation of Trender

Being curious about Trender's performance, I searched for polygon representations of my country, and eventually found the GTOPO30 dataset. Using these data, I coded a real-time "flight-simulation" over South Greece (including the Aegean Islands and Rhodes).

On my Athlon XP 1600 (1,4GHz) with a Radeon 9600, I get 35 frames per second for the 2Km map (at 107000 triangles per frame!)

Download:

  • Win32 demo by clicking 4Km (350KB) or 2Km (1.2MB).
    The first one has sampled the terrain every 4000 meters, the second one every 2000 meters (if you have a fast CPU/Card, I suggest the 2Km one).
    Needless to say, if you don't have a fairly decent OpenGL card, be prepared for really slow frame rates (107000 triangles per frame...)
  • Linux demo here
    You still need one of the above Win32 packages for the map data!
    To run it, make sure you have hardware accelerated OpenGL in your X server and SDL installed.
